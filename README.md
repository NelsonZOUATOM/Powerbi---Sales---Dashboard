## 🎯 Project Objective

Analyze sales and cancellation data to uncover trends by product category, region, and time period — and turn raw data into actionable business insights.

---

## ⚙️ Features Built

- 📐 **Data Modeling** — Structured relational data model
- 🧮 **DAX Measures** — Custom KPIs and calculated metrics
- 📊 **Histograms** — Distribution analysis by category
- 📉 **Cancellation Rate (%)** — Month-over-month 
  evolution tracking
- 🗂️ **Multi-tab Navigation** — Dedicated pages for 
  Sales and Cancelled Orders
- 🔍 **Dynamic Filters** — Slicers by date, region, 
  category and product
- 💬 **Custom Tooltips** — Advanced info-bubbles on hover
- 🔖 **Bookmarks** — Quick navigation between views
- 🔐 **Row-Level Security** — Role-based data access

---

## 📈 Key Insights

- ❌ Q3 records the highest cancellation rate
- 📱 Smartphones & laptops: top sellers but also most cancelled
- 🌍 Regional disparities significantly impact overall performance
